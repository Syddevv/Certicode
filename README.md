# Certicode

Certicode is an open-source **e-commerce web application** built with **React and Vite**. The project demonstrates a modern frontend architecture for building fast and scalable online shopping platforms.

This project was developed collaboratively by a team of developers to explore real-world web development practices such as component-based design, state management, and responsive UI development.

---

## 🚀 Features

* Product browsing interface
* Shopping cart functionality
* User authentication interface
* Order management interface
* Responsive design for mobile and desktop
* Modern React component architecture

---

## 🛠️ Built With

* **React** – Frontend UI library
* **Vite** – Fast development build tool
* **JavaScript (ES6+)**
* **HTML5 & CSS3**

---

---

## ⚙️ Installation

To run the project locally:

```bash
git clone https://github.com/Syddevv/Certicode.git
cd Certicode
npm install
npm run dev
```

Then open the development server shown in the terminal.
